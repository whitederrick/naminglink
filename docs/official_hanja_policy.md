# 인명용 한자 정책 메모

서비스에서 한자 후보를 추천할 때는 다음 원칙을 UI와 프롬프트에 반영한다.

1. 최종 법적 등록 가능 여부는 대한민국 전자가족관계등록시스템 또는 대법원 인명용 한자 기준으로 확인해야 한다.
2. 서비스는 후보 추천, 의미 해석, 배제 사유 설명을 제공하지만 법적 확정 판정으로 표현하지 않는다.
3. 배제 사유는 최소한 다음 범주로 설명한다.
   - 이름에 쓰기 부정적인 직접 의미
   - 인명용 한자 공식 기준 확인 필요
   - 발음/의미 충돌
   - 성씨와 결합했을 때 어감 또는 획수 부담
   - 부모가 입력한 제외 조건과 충돌
4. 추후 실제 인명용 한자 데이터셋을 연결하면 후보 생성 전 필터링과 결과 후 검증을 분리한다.

초기 구현은 공식 데이터셋을 내장하지 않고, AI 응답에 `official_verification_note`를 포함하도록 강제한다.
