## 오늘 최종 완료한 변경

### 안덕규 후보 정합성

- 돌림자 `규(奎)`는 모든 후보에서 고정한다.
- `徳 · 德의 略字`처럼 본자의 약자 설명만 있는 글자는 독립 의미 후보로 추천하지 않는다.
- `㯖 · 덕,적`처럼 뜻풀이가 아니라 음가 목록만 있는 글자는 추천하지 않는다.
- `悳`처럼 이미 남은 후보와 뜻이 같은 글자는 의미 중복 후보로 정리한다.
- 제공 데이터 기준으로 안덕규의 독립적이고 설명 가능한 결과는 `德奎` 1개이다.
- 후보가 부족한 경우 억지로 5개나 10개를 채우지 않는다.

### 상품·화면·PDF 후보 수 일치

- 상품명과 가격 안내를 고정 `5개/10개`가 아닌 `최대 5개/최대 10개`로 수정했다.
- 결제 패널에 현재 결과에서 실제 제공되는 후보 수를 함께 표시한다.
- PDF 종합 비교 제목과 AI 지시문은 실제 후보 수를 사용한다.
- 화면 후보가 PDF 후보 변환 과정에서 누락되면 조용히 축소하지 않고 오류로 감지한다.
- 안덕규 회귀 테스트에서 화면 1개, PDF 데이터 1개, 후보 상세 분석 1개가 모두 `德奎`로 일치함을 확인했다.
