- 관리자 메뉴는 랜딩에서 제거하고, `/admin`은 별도 직접 접근 구조로 유지했습니다.

  - 한글 이름을 글로벌 이름으로 변환

  - 외국 이름을 한국 이름으로 변환

- 국가 선택 데이터를 30개국 기준으로 확장했습니다.
  - 일본, 베트남, 필리핀, 태국, 인도네시아, 중국, 대만, 우즈베키스탄, 몽골, 인도, 미국, 멕시코, 브라질, 프랑스, 영국, 튀르키예, 러시아, 아랍에미리트, 이집트, 사우디아라비아, 캄보디아, 말레이시아, 독일, 호주, 캐나다, 아르헨티나, 칠레, 카자흐스탄, 폴란드, 이탈리아
- 국가 선택 시 기본 결과 언어가 자동으로 설정되도록 했습니다.
- 베트남, 인도네시아, 태국은 특별 분기 강제가 아니라 `추천 목적 옵션`으로만 반영했습니다.
  - 베트남: 한국어 교육, 취업, 유학 실용 목적 추천
  - 인도네시아/태국: K-culture, SNS, 부캐, 애칭 이름 목적 추천

### 3. 서비스 입력 폼

- `외국 이름 -> 한국 이름` 폼에서 국가를 선택하면 국가별 기본 언어와 현지 이름 예시를 안내합니다.
- `한국 이름 사용 목적` 옵션을 추가했습니다.
  - 국가별 추천 목적 자동 선택

- 분석 전 이용약관 및 개인정보처리방침 동의 체크를 추가했습니다.
