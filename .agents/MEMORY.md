# Naming-Link 작업 메모리

마지막 갱신: 2026-07-19

## 현재 운영 상태

- 운영 URL은 `https://naminglink.vercel.app`이며 이번 최종 변경을 `main`에 커밋해 운영 배포한다.
- 프리미엄 상품은 실제 후보 확보 수에 따라 제공하며 `최대 5개/최대 10개`로 표시한다.
- 결제 화면은 현재 제공 후보 수를 표시하고 화면·PDF·후보 상세 분석의 후보 수와 순서를 강제로 일치시킨다.
- 안덕규 + 돌림자 `奎` 회귀 결과는 `德奎` 1개이다.
  - `徳`: 본자 `德`의 약자 설명뿐이므로 독립 후보 제외
  - `㯖`: `덕,적`은 뜻이 아닌 음가 목록이므로 제외
  - `悳`: `德`과 의미가 같아 중복 정리
- 돌림자 강제 고정, PUA·부정 의미·성별 조건 충돌·불충분 의미 필터와 한국 국자 배지를 적용했다.
- 결과 화면의 `입력 수정`은 서비스 입력 URL로 이동하며 직전 입력값을 복원한다.
- 로컬 개발 환경은 무료 일일 한도를 차감하지 않는다.
- 로컬 서버는 포트 `3001`이며 사용자가 직접 관리한다. 임의로 시작·종료하지 않는다.
